# Gridcode

> **A universal location language that turns any point on Earth into a short, human‑readable code — easy to say, easy to type, impossible to confuse.**

Gridcode replaces traditional latitude/longitude with a **structured**, **scalable**, and **psychologically intuitive** system.  
The code grows in precision as you add characters, so the same identifier can describe a country‑scale region, a neighbourhood, or a 1‑metre square — all without separators or symbols.

It’s fast, compact, and globally consistent.  
If you can read a postcode or a bank card number, you can read a Gridcode.

**In short: Gridcode makes the entire planet addressable, searchable, and shareable with a single, elegant system.**
